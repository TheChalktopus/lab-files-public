Apply Kirchoff's Laws

Voltage V(x) = R'dx I(x) + i ω L' dx I(x) + V(x + dx) -> [V(x + dx) - V(x)]/dx = \diff{V}{x} = -(R' + i ω L)I(ω)

Current I(x) = I(x + dx) + V(x + dx)[i ω C' dx] -> [I(x + dx) - I(x)]dx = -V(x)[G' + i ω C']

Result: Telegrapher's equations

\pdiff{V}{x} = -(R' + i ω L')I, \pdiff{I}{x} = -(G' + jω C')V

Coupled diffy Q's: \pdiff{^2V}{x^2} = -(R' + i ω L')\pdiff{I}{x} = (R' + i ω L')(G' + i ω L') V γ = \sqrt{(R' + i \omega L')(G' + i \omega C')} γ called "Complex Propagation constant" \pdiff{^2 V}{x^2} = γ2 V \pdiff{^2 I}{x^2} = γ2 I

Solve these diffy Q's V(x) = V0^+ e-γ x + V0-e-γ x I(x) = I0^+ e-γ x + I0-e-γ x Waves! (Recall γ is complex)

\pdiff{V}{x} = -(R' + i ω L')I(x) = -γ V0+e-γ x + γ V0-eγ x Pull out a common factor of γ I(x) = γ/(R' + i ω L') [V0^+ e-γ x - V0e-γ x]

γ/(R' + i ω L') is our admittance or 1/Impedance Z. "Characteristic Impedance" Z0 = (R' + i ω L')/γ = \sqrt{\frac{R' + i \omega L}{G' + i \omega L}} NB: This relates V0^+ and I0^+

V(x) = V0^+ e-γ x + V0^- e^γ x I(x) = \frac{1}{Z_0}(V0^+ e-γ x - V0^- eγ x)

γ = α + i β α -> "Attenuation constant" (decay!) β -> "phase constant" ^ see these in a datasheet

Low-loss Line ie. a real line (we have losses in real life)

R ≪ ω L G ≪ ω C

Rederive important steps. Find phase constant and attenuation constant, β and α

To first order: \[\gamma = i \omega \sqrt{L'C'}(1 - i R'/ (\omega L'))^(1/2)(1 - i G'/(\omega C'))^(1/2)\] \[\gamma = i \omega \sqrt{L'C'}(1 - i R'/(2\omega L') - i G'/(2\omega C') - \mathcal{O}(R'^2))\] \[\alpha = \frac{1}{2} R' \sqrt{\frac{C'}{L'}} + G'\sqrt{\frac{C'}{L'}}\] \[\beta = \omega \sqrt{L'C'}\] Still travel at \(c\)(!)

Characteristic impedance Z0 = \sqrt{R' + i \omega L'}/\sqrt{G' + i \omega C'} = \sqrt{L'/C'} to zeroeth order

Can then rewrite \(\alpha = \frac12 (R'/Z_0 + G' Z_0)\). Attenuation!
